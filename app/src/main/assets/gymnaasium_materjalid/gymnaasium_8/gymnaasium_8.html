Astmefunktsioon

Funktsioone kujul y = xa,a ∈ R nimetatakse astmefunktsioonideks. Tüüpilised astmefunktsioonid on

  • a = 0, sellisel tegemist on konstantse funktsiooniga y = 1.
  • a = 1, sellisel juhul on tegemist lineaarse funktsiooniga y = x.
  • a = -1, sellisel juhul on tegemist pöördvõrdelise funktsiooniga y = .
  • a = 2, sellisel juhul on tegemist ruutfunktsiooniga y = x2.
  • a = 3 on kuupfunktsioon kujul y = x3. Kuupfunktsiooni määramispiirkond on terve reaalarvude hulk R.
  • Funktsioonid kujul y = x2n,n ∈ Z+ ehk positiivse paarisarvulise astendajaga arvud.. Määramispiirkonnaks on kogu reaalarvude hulk ning muutumispiirkonnaks on kõikide mittenegatiivsete arvude hulk [0;∞). Positiivsuspiirkonnaks on R\{0} ning negatiivsusvahemikud puuduvad. Funktsiooni nullkohad on X0 = {0}. Kasvamisvahemikuks on vahemik [0;∞) ja kahanemisvahemikuks (-∞,0].
  • Funktsioonid kujul y = x2n-1,n ∈ Z+ . Määramis- ja muutumispiirkondadeks on sellisel kujul kogu reaalarvude hulk R. Positiivsuspiirkond on (0;∞), negatiivsuspiirkond on (-∞,0) ja funktsioon kasvab terve määramispiirkonna ulatuses. Kahanemispiirkonnad puuduvad.

Juurfunktsioon

Funktsiooni kujul y = nimetatakse ruutjuureks. Ruutjuure määramis- ning muutumispiirkonnaks on vahemik [0;∞]. Ruutjuur on kogu määramispiirkonnas pidev ning kasvab rangelt.

Funktsiooni kujul y = nimetatakse kuupjuureks. Kuupjuure määramis- ning muutumispiirkonnaks on vahemik [-∞;∞]. Funktsioon on kogu piirkonnas rangelt kasvav ning pidev. Funktsioon on paaritu ning tema graafik on sümmeetriline nullpunkti suhtes.

Funktsioon kujul y = on määratud vahemikus [0;∞] ning samuti tema muutumispiirkond on [0;∞]. Funktsiooni graafik on kogu ulatuses kumer.

Funktsioon kujul y = on määratud vahemikus [-∞;∞] ning samuti tema muutumispiirkond on [-∞;∞]. Funktsiooni on kogu määramispiirkonnas pidev ja rangelt kasvav. Funktsiooni graafik on nõgus, kui x < 0 ning kumer kui x > 0.

Eksponentfunktsioon

Funktsiooni kujul y = cax, kus a > 0,a≠1 ja x ∈ R, nimetatakse eksponentfunktsiooniks. Eksponentfunktsioonide määramispiirkonnaks on hulk R = [-∞;∞].

Omadused:

  • Eksponentfunktsiooni positiivsuspiirkond on kogu tema määramispiirkond st. hulk R. Eksponentfunktsioonil puudub negatiivsuspiirkond.
  • Eksponentfunktsioonil puuduvad nullkohad.
  • Funktsioon läbib graafikul punkti (0;1).
  • Kui a > 1, siis on funktsioon kasvav. Kui 0 < a < 1, siis on funktsioon kahanev.
  • Kui a > 1, siis on funktsiooni väärtused kasvavad argumendi kasvamisega lõpmatult suurelks. Kui 0 < a < 1, siis funktsiooni tulemus hakkab argumendi kasvades lähenema nullile.
  • Funktsioonide y = ax ja y = ()x graafikud on sümmeetrilised y-telje suhtes.

Logaritmfunktsiooniks nimetatakse funktsiooni kujul y = log ax, kus a > 0 ja a≠1. Sellisel kujul funktsioon on pöördfunktsiooniks eksponentfunktsioonile kujul y = ax. Logaritmfunktsiooni määramispiirkonnaks on kogu positiivsete reaalarvude hulk {x ∈ R,kusx > 0}. Muutumispiirkonnaks on kogu reaalarvude hulk R.

Logaritmfunktsiooni omadused:

  • Juhul, kui a > 1:
    • Ainsaks nullkohaks on x0 = 1. Samuti läbib graafik punkti (1;0).
    • Positiivsuspiirkond on 1 < x < ∞. Negatiivsuspiirkond on vahemik 0 < x < 1.
    • Funktsiooni kasvamispiirkond on 0 < x < ∞.
    • Funktsiooni argumendi kasvamisel hakkab funktsiooni tulemus lähenema lõpmatusele.
    • Kui funktsiooni argument läheneb nullile (paremalt), hakkavad funktsiooni väärtused lähenema miinus lõpmatusele.
  • Juhul 0 < a < 1:
    • Ainsaks nullkohaks on x0 = 1. Samuti läbib graafik punkti (1;0).
    • Positiivuspiirkond on 0 < x < 1. Negatiivsuspiirkond on vahemik 1 < x < ∞.
    • Funktsioon kahaneb kogu määramispiirkonnas.
    • Funktsiooni argumendi kasvamisel hakkab funktsiooni tulemus lähenema miinus lõpmatusele.
    • Kui funktsiooni argument läheneb nullile (paremalt), hakkavad funktsiooni väärtused lähenema pluss lõpmatusele.
