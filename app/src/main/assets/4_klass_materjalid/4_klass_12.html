Jagamistehe

  • a : b = c
  • a = b ⋅ c
  • b = a : c

Jagamistehtes oleva tundmatu leidmiseks tuleb käituda analoogselt lahutamisega. Kui tundmatu on jagatav, siis saame selle leida jagaja ning jagatise korrutisena. Näiteks kui x : 3 = 4, siis x = 3 ⋅ 4 = 12. Kui tundmatu on jagaja, siis jagame jagatava jagatisega. Näiteks kui 15 : x = 5, siis x = 15 : 5 = 3.
